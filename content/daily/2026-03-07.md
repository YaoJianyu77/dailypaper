# Daily Paper Report - 2026-03-07

## Overview
今天最值得读的主线不是再造更大的模型，而是更接近真实系统瓶颈的两类工作：一类处理 LLM 的可信性与部署成本，包括无检索事实核查和小模型证据归因；另一类直接解决训练与推理效率，包括推测解码词表裁剪和更省内存的稳定训练。多模态图推理与单流语音语言建模则提供了两个值得持续跟踪的问题扩展方向。建议先读 2603.05471v1 和 2603.05210v1，再看 2603.05181v1，最后按训练方法与垂直场景补齐。

## Top Themes
- LLM 可信性：无检索事实核查与证据归因
- 系统效率：推测解码词表裁剪与省内存稳定训练
- 模态扩展：多模态图推理与单流语音语言建模

## Reading Strategy
- 先读 2603.05471v1，判断无检索事实核查是否能改变 agent 可信性链路。
