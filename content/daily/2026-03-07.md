# Daily Paper Report - 2026-03-07

## Overview
今天最值得读的主线有两条：一条是把 LLM 的可信判断能力从检索依赖里拆出来，另一条是把多模态图推理与训练推理效率继续推向更可落地的系统设计。优先读 2603.05471 和 2603.05181，前者直接对应 Agent 可信性，后者则把多模态推理从孤立图文对推进到带拓扑约束的图结构；随后读 2603.05500 和 2603.05210 判断效率论文的证据是否足够硬，最后再看 2603.05308 与 2603.05299，评估专域小模型验证和单流语音 LM 这两条延展路线的真实上限。

## Top Themes
- 无检索事实核验开始把 LLM 参数知识本身当作可直接利用的验证资源
- 多模态推理正在从图文对理解转向带关系结构的图级建模
- 训练与推理效率论文继续围绕显存、吞吐和草稿模型延迟这些硬瓶颈展开竞争式优化

## Reading Strategy
- 先读 2603.05471，重点看无检索事实核验的任务定义、评测框架和 INTRA 是否真的优于基于 logit 的做法。
- 再读 2603.05181，核对图条件 VLM 与模态路由是否真正解决了跨模态一致性和异质模态偏好。
- 工程导向读者接着看 2603.05500 与 2603.05210，分别关注训练显存与推理延迟的硬证据。

- Domain: Multimodal
- Reading priority: high
- Authors: Yuanfu Sun, Kang Li, Pengkang Guo, Jiajin Liu, Qiaoyu Tan

- One-line view: 这篇工作把多模态推理对象从孤立图文对换成带边结构的多模态图，并用可学习路由决定每个节点该看哪种模态组合。
- Why now: 与多模态和 LLM 主线高度契合，问题设置比普通图文任务更接近真实系统；但实验摘要不完整，需要带着证据意识去读。
- Core contribution: 把多模态图推理中的两个核心障碍明确为跨模态一致性弱和模态偏好异质。

- Why read: 它直接对应多模态 LLM 下一步常见痛点：输入不再是单轮图文，而是带关系结构的复杂对象。
- Why read: 方法里同时有表示学习和指令调优两层设计，值得看两者分工是否清晰。
- Why read: 如果你关心多模态 Agent 读图、读文档、看关系图，这篇论文的问题设置比普通 VLM 论文更接近真实系统。
- Risk: 摘要截断导致实验结论不完整，当前无法判断收益是否稳定。
- Risk: 方法链路较长，训练和推理成本可能不低，工程可复现性需要核对。
- Recommended for: 做多模态 LLM 与图推理的研究者, 关注文档图、知识图或复杂实体关系理解的工程师, 想研究输入路由与异构模态选择机制的读者

- Recommended for: 做 Agent 可信性与事实核验的研究者, 关注 LLM 内部表征与参数知识利用的研究者, 设计低延迟审核或离线验证系统的工程师
