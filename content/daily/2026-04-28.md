### 1. Long-Context Aware Upcycling: A New Frontier for Hybrid LLM Scaling

*arXiv preprint | 2026-04-27T17:23:37Z | Parsa Ashrafi Fashi, Utkarsh Saxena, Mehdi Rezagholizadeh, Aref Jafari et al.*

> HyLo upcycles pretrained Transformer LLMs into hybrid long-context models using efficient attention and linear sequence blocks to cut KV-cache memory while extending usable context.

Hybrid sequence models can reduce long-context inference costs, but training them from scratch wastes the value of existing Transformer checkpoints. This paper proposes HyLo, an upcycling recipe that adapts pretrained Llama- and Qwen-style models with efficient Transformer blocks, Multi-Head Latent Attention, Mamba2 or Gated DeltaNet blocks, staged long-context training, and teacher-guided distillation. The abstract claims up to 32x longer usable context, more than 90% KV-cache memory reduction, and vLLM prefill/decoding up to 2M tokens, while comparable Llama baselines run out of memory beyond 64K context. It also reports strong short- and long-context results at 1B and 3B scale, including RULER and comparisons against upcycled hybrid baselines and JetNemotron. The main missing detail is the full evaluation setup: hardware, latency and throughput behavior, baseline implementation fairness, and how much quality is preserved outside the reported tasks.

If the claims hold, the method offers a practical post-training route to reduce KV-cache pressure and push long-context serving further without pretraining a new model from scratch. A key caveat is that the abstract gives memory and benchmark claims but omits latency, throughput, and hardware assumptions. Another caveat is that baseline fairness is unclear without details on implementation, context settings, and training data exposure fairness for JetNemotron comparisons alike; missing implementation details may determine whether the memory gains translate to deployable serving gains.

### 2. DepthKV: Layer-Dependent KV Cache Pruning for Long-Context LLM Inference

*arXiv preprint | 2026-04-27T16:15:37Z | Zahra Dehghanighobadi, Asja Fischer*

> DepthKV treats KV-cache pruning as a layer-wise budget allocation problem instead of applying the same pruning ratio to every layer.

Long-context LLM inference is constrained by KV-cache memory that scales linearly with sequence length. The paper argues that uniform layer-wise KV pruning is wasteful because layers differ in their sensitivity to cached-token removal. DepthKV keeps a fixed global KV budget but reallocates it across layers according to pruning sensitivity. The abstract claims consistent gains over uniform pruning at the same global pruning ratio across multiple models and tasks, suggesting better use of the cache budget rather than a larger memory allowance. The missing details are the sensitivity-estimation procedure, the exact workloads and models, and whether the method improves real serving metrics such as latency, throughput, and memory on specific hardware.

If the sensitivity allocation is cheap and robust, the method could preserve more model quality under the same KV memory budget for long-context inference. A key caveat is that the abstract does not specify the sensitivity-estimation algorithm, making implementation cost unclear. Another caveat is that evaluation details are underspecified, including model sizes, context lengths, tasks, and hardware assumptions.

### 3. Kwai Summary Attention Technical Report

*arXiv preprint | 2026-04-27T12:59:53Z | Chenglong Chu, Guorui Zhou, Guowang Zhang, Han Li et al.*

> Kwai Summary Attention proposes learnable summary tokens to compress historical long-context state at an O(n/k) ratio, targeting KV-cache and attention-cost pressure without fully abandoning distant-context retention.

This technical report targets the long-context inference and training cost created by standard softmax attention, where sequence length drives quadratic attention work and linearly growing KV cache. It positions Kwai Summary Attention as an intermediate path between per-layer KV compression and KV-friendly architectures such as local or linear attention: keep a linear dependence on sequence length, but compress historical context semantically by a ratio k. The proposed mechanism is to turn past context into learnable summary tokens, aiming to reduce sequence modeling cost while retaining long-distance dependency information in a more referential and interpretable form. The abstract makes the systems motivation clear, especially around KV cache and long-context overhead, but it does not provide experimental results, hardware setup, latency or memory numbers, or baseline comparisons. The main missing detail is whether the summary-token mechanism preserves quality and actually improves end-to-end serving efficiency under realistic long-context workloads.

If validated, the method could offer a practical long-context inference trade-off: lower context-state cost while preserving more distant dependency information than aggressive cache reduction or locality-heavy attention. A key caveat is that the abstract contains no quantitative evidence, so the claimed cost reduction is not yet tied to measured serving behavior. Another caveat is that the mechanism is underspecified: summary-token construction, update frequency, and integration with standard attention are unclear from the abstract alone.

### 4. Diffusion Templates: A Unified Plugin Framework for Controllable Diffusion

*arXiv preprint | 2026-04-27T11:44:45Z | Zhongjie Duan, Hong Zhang, Yingda Chen*

> A systems-level plugin interface for controllable diffusion that standardizes how task-specific control representations are cached, merged, and injected into diffusion runtimes.

Controllable diffusion has become fragmented across backbone-specific pipelines, parameter formats, and runtime hooks, making reuse and composition hard. This paper proposes Diffusion Templates, a plugin framework that separates base-model inference from capability injection through Template models, a standardized Template cache, and a pipeline for loading, merging, and injecting one or more controls. The abstract claims the same abstraction can cover heterogeneous carriers such as KV-Cache and LoRA, and reports a model zoo across structural control, editing, enhancement, inpainting, reference, and age-control cases. Its practical value is in runtime modularity and composability for diffusion systems, not in LLM inference acceleration or kernel-level optimization. The abstract does not provide latency, throughput, memory, baseline, or ablation details, so the systems payoff remains underspecified.

If the abstraction works across backbones and control carriers, it could reduce integration cost for diffusion-control systems and make multi-control composition easier to maintain. A key caveat is that the abstract gives no quantitative evidence on runtime overhead, memory cost, or quality tradeoffs. Another caveat is that the framework may hide backbone-specific complexity rather than eliminating it; the abstract does not clarify integration burden.
